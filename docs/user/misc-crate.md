# The Misc crate

The crate signed `Misc` is the catch-all. After every other crate has
taken its items, whatever is left goes there. If you have no Misc
crate, unmatched items simply stay in your backpack; nothing is ever
lost.

Make one early. It is the difference between "everything has a place"
and "why is this still in my bag".

## Renaming it

The name comes from `StowItConfig.xml`:

```xml
<FallbackCrateName>Misc</FallbackCrateName>
```

Change `Misc` to any word you like, in any language, then sign the
crate with that word. Save the file and type `stow reload` in the F1
console.
