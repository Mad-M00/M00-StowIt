Installation

StowIt is a DLL (code) mod. Installing it is a two-minute job, but the details below matter — most "it doesn't work" reports come down to one of these steps.

1. Download

Grab the latest release zip from GitHub Releases.

2. Find your Mods folder

The game reads mods from two places. Either works — pick one and stay consistent:

  • Game install folder (the classic spot):
    ...\Steam\steamapps\common\7 Days To Die\Mods
  • User data folder (the spot The Fun Pimps recommend): press Windows + R, type %APPDATA%\7DaysToDie\Mods and hit Enter.

If the Mods folder doesn't exist yet, create it — the name is exactly Mods.

3. Extract the zip

Unpack so the folder structure looks like this:

Mods\
  M00-StowIt\
    M00StowIt.dll
    ModInfo.xml
    CrateLabels.txt
    ...

The most common mistake: extracting a folder into a folder, ending up with Mods\M00-StowIt\M00-StowIt\ModInfo.xml. If ModInfo.xml is not directly inside Mods\M00-StowIt, the game will not see the mod.

4. Turn off EAC (anti-cheat)

Like every code mod, StowIt only loads with EAC off:

  1. Start the game via the launcher (right-click the game in Steam > Show launcher, or just start it normally if the launcher appears).
  2. Untick Use EAC (Easy Anti-Cheat).
  3. Start the game from that launcher.

With EAC left on, nothing breaks — the mod simply skips itself (SkipWithAntiCheat) and you'll wonder why nothing happens.

5. Check the game's own Harmony folder

Look inside the game install's Mods folder (the steamapps one, even if you put StowIt in AppData) for a folder named 0_TFP_Harmony. It ships with the game and every DLL mod needs it.

Missing? People often delete it when clearing out old mods. Get it back in Steam: right-click 7 Days to Die > Properties > Installed Files > Verify integrity of game files.

6. Confirm it loaded

Start a world, press F1 and type stow. If the console answers with the command help, you're done — sign some crates and press LeftAlt + X. If it says "unknown command", walk back through steps 3–5.

Installing on an existing save

Go right ahead — no new game needed, and your containers can keep whatever names they already have. StowIt never writes into your save files: it reads crate signs only at the moment you press the sort key, and any sign text it does not recognise is simply ignored. Label the crates you want sorted into whenever it suits you — there is nothing to rename or empty first.

Multiplayer and dedicated servers

Every player needs the mod installed — as a DLL mod it does not push itself from the server. On a dedicated server, install the same version into the server's Mods folder too, and keep versions matched between server and clients.

Updating

  1. Close the game (the DLL can't be swapped while it runs).
  2. If you customised CrateLabels.txt, copy it somewhere safe.
  3. Delete the old Mods\M00-StowIt folder and extract the new zip.
  4. Copy your customised CrateLabels.txt back over the new one.

Uninstalling

Delete the Mods\M00-StowIt folder. That's all. The mod never touches game files or your save — everything it sorted stays exactly where it is.
